Author

Executive Summary

Netflix continues to dominate the global streaming market with its diverse portfolio of original content. Through analysis of Netflix’s public Top 10 data, this report identifies key successes and growth opportunities across different markets and content categories. Our findings highlight the remarkable global reach of shows like Stranger Things, the growing importance of regional content in markets like India, and emerging trends in international programming preferences.

Data Formatting Functions

# Function to format column titles
format_titles <- function(df){
    colnames(df) <- str_replace_all(colnames(df), "_", " ") |> str_to_title()
    df
}

# Function to create publication-ready tables
create_pub_table <- function(df, n_rows = 20) {
    df |>
        head(n = n_rows) |>
        datatable(options = list(searching = FALSE, info = FALSE, pageLength = 10)) |>
        formatRound(c('Weekly Hours Viewed', 'Weekly Views'), digits = 0)
}
